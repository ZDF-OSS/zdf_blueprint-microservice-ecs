## Getting Started 
* Install the aws cli
* Install aws cdk

## Running 
* Clone the repository
* Run the following to deploy

```  
chmod +x ./scripts/deploy.sh
./scripts/deploy.sh
``` 

* Run the following to destroy the solution
```  
chmod +x ./scripts/cleanup.sh
./scripts/cleanup.sh
```
